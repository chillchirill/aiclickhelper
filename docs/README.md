# Documentation

This folder explains the project at two levels:

1. `python-mini-intro.md`
   A beginner-friendly guide to the Python syntax and object-oriented patterns used in this project.
2. `architecture-overview.md`
   A high-level explanation of the application architecture and module responsibilities.
3. `runtime-flow.md`
   A step-by-step walkthrough of what happens from user prompt to overlay recommendation to auto-advance.
4. `file-guide.md`
   A practical map of the codebase with notes about what each file is responsible for.

Recommended reading order:

1. `python-mini-intro.md`
2. `architecture-overview.md`
3. `runtime-flow.md`
4. `file-guide.md`
